Dendron

Setting up Dendron

  1. Install Visual Studio Code
  2. Install git (Windows only)
    1. Add git to Windows PATH
  3. Install Dendron
    1. Open the command palette using Cmd/Ctrl + Shift + P
    2. Create a new workspace using > Dendron: Initialize Workspace
  4. Make a GitHub account
    1. Send username to @wrdwllm (Private)
    2. Accept invitation to the winter-intensive repository
  5. Link GitHub to VS Code using the GitHub extension
  6. Install required extensions
    1. markdownlint
    2. Dendron Markdown Shortcuts
    3. GitHub Pull Requests
    4. Dendron Paste Image
  7. Initialise a remote Dendron vault using the GitHub link
    1. Run > Dendron: Add Vault
    2. Select remote
    3. Paste https://github.com/wrdwllm/winter-intensive.git and press Enter
  8. Add content and sync with remote repository
    1. Pull most recent commit when you open Dendron
      1. Open terminal shell
      2. Navigate to root workspace directory with .git (not vault)
      4. Run git pull
    2. Push to remote after making significant updates
      1. git add .
      2. git commit -m 'commit message
      3. Resolve conflicts using the Source Control tab (contact @wrdwllm (Private) if unsure)
      4. git push

Workflow

  1. Follow the Getting Started Guide and the Dendron tutorial
  2. Use Markdown
  3. Make a new note
    1. Press Cmd/Ctrl + L
    2. Write content
    3. Link to a note using syntax [[note.file.name]]
    4. Open Markdown preview window using > Dendron: Show Preview
    5. Use GitHub Flavoured Markdown
    6. Resolve display issues with > Dendron: Reload Index and > Developer: Reload Window
  4. Sync to repository (see Step 8 in previous section)
